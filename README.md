# Weather-api
Api rest with node and express

# Features
  * Express
  * Rest API
  
# Requirements
  * node & npm

# Installation
  * git clone https://github.com/MarcosRissottoB/weather-api.git
  * cd weather-api
  * npm install or npm i
  
# Run project
  * npm start

# GET Routes
  * visit http://localhost:5000/api/v1
      /location
      /current?city
      /forecast
